100-GRID is a basic and simplistic grid system.

100-GRID is aimed at teaching students the basics of using CSS grids and how powerful and useful they can be to any skill leveled web developer!

The aim is to also bring an understanding to the system and teach students how to develop their own grids.

View RAW CSS

How does 100-GRID work?

100-GRID works by using percentage based sizes that go up in increments of five percent for each size. For example you would have 'grid-05','grid-10' and so on...

All the grids are aligned with 'inline-block' and have a negative 'margin-right' of four pixels. This ensures all the grids line up equally.

Because the grid revolves around increment of five percentages, it's realtively easy to space and work the grids out. The grids are also nestable and take in there parents height values so spacing of elements inside the grids are a lot easier to control individually.

This grid system was also made for fixed widths in mind or a complete one hundred percent wide website.

Why did I include normalize.css?

I chose to include this style sheet directly into the grid because it's a preference I have been using for my own web development adventures ever since I started.

I believe it's an incredibly powerful tool. 100-GRID was built around normalize but that doesn't mean you need to use it! This grid system works perfectly fine without it.

Removing it is as simple as deleting the minified line from the grid style sheet or commenting it out!
